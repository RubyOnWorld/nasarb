= fortran

* https://github.com/kleb/nasarb/tree/master/fortran
* https://rubygems.org/gems/fortran

== DESCRIPTION:

Fortran utilities.

Currently, only provides Fortran 90/95/2003 dependencies.

== SYNOPSIS:

  require 'fortran'

  Fortran::Dependencies.new( 'main.f90' ).dependencies

== REQUIREMENTS:

* Ruby

== INSTALL:

* <tt>sudo gem install fortran</tt>
